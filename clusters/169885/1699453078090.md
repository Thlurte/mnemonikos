### Random Number Generator
`srand()` method is used to set seed for the randomized number. And `rand()` could be used to generate a random number between --.

```cpp
#include <iostream>
#include <ctime>
int main(){
    srand(time(NULL));

    int dice1 = (rand() % 6)+1;
    int dice2 = (rand() % 6)+1;
    int dice3 = (rand() % 6)+1;
    std::cout << dice1 << '\n';
    std::cout << dice2 << '\n';
    std::cout << dice3 << '\n';
    return 0;
}
```
